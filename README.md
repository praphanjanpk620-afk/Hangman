Hangman

A simple Hangman game built in Python and compiled into a Windows executable.

- Run the `main.exe` file from the Releases page
- Guess letters to reveal the hidden word
- You lose after too many wrong guesses

 [Download the latest release](https://github.com/praphanjanpk620-afk/Hangman/releases)


- The program picks a random word
- You guess letters one at a time
- The game tracks wrong guesses and displays progress

  
- Python 3
- Compiled using PyInstaller (or whatever you used)
